## Setup



### Python

Using python 3.9

```shell
python -m venv venv
. venv/bin/activate
python -m pip install air-sdk
export AIR_USER='' #change to air email
export AIR_TOKEN='' #generate an air token
```

### Direnv (optional)

To use direnv, edit `.envrc` and add your email and token then `direnv allow`
